# Images Directory

Place your image assets in this directory.

## Required Images

### Logo
- **File**: `logo.svg` or `logo.png`
- **Recommended Size**: 180x60px (3:1 ratio)
- **Format**: SVG preferred for scalability
- **Usage**: Header, footer

### Hero/Background (Optional)
- **File**: `hero-background.jpg` or `hero-background.webp`
- **Recommended Size**: 1920x1080px
- **Format**: WebP with JPG fallback
- **Usage**: Hero section background

### Water Heater Image (Optional)
- **File**: `water-heater.jpg`
- **Recommended Size**: 800x600px
- **Format**: WebP with JPG fallback
- **Usage**: Product showcase sections

### Tom's Mascot (Optional)
- **File**: `tom-mascot.svg` or `tom-mascot.png`
- **Recommended Size**: 400x400px
- **Format**: SVG or PNG with transparency
- **Usage**: Hero section, testimonials

### Social Sharing Image
- **File**: `og-image.jpg`
- **Recommended Size**: 1200x630px
- **Format**: JPG
- **Usage**: Open Graph social media previews

## Image Optimization Tips

1. **Use WebP format** for modern browsers with JPG/PNG fallback
2. **Compress images** to < 100KB for hero images
3. **Use SVG** for logos and icons when possible
4. **Add alt text** in the HTML for accessibility
5. **Implement lazy loading** for below-fold images

## Tools for Optimization

- [TinyPNG](https://tinypng.com/) - Compress JPG/PNG
- [Squoosh](https://squoosh.app/) - Convert to WebP
- [SVGOMG](https://jakearchibald.github.io/svgomg/) - Optimize SVG
